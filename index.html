核心特性

  • 🚀 **快速调用:** 自定义全局热键，随时随地唤出片段列表。
  • 📂 **分组管理:** 支持多级分组，清晰整理您的文本片段。
  • 🖱️ **一键复制:** 双击即可将所需文本复制到剪贴板。
  • 🖥️ **后台运行:** 安静驻留系统托盘，不干扰您的工作。
  • ⚙️ **可视化面板:** 直观的管理界面，轻松增删改查。
  • ✨ **现代界面:** 基于 WebView2 的美观 UI (可选)。
  • 🔌 **功能集成:** (可选) 内置窗口拖拽、屏幕键盘等实用工具。

如何使用？

  1. 下载并安装 Snippet Butler。
  2. 通过系统托盘图标打开管理面板，添加您的文本片段和分组。
  3. 设置一个您喜欢的全局快捷键。
  4. 在任何需要输入的地方，按下快捷键，双击选中片段即可复制！
